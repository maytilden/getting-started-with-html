GETTING STARTED WITH THE WEB AND HTML

Web Fundamentals

Html is the main type of document/content on the web. When a person uses his or her browser to go to a webpage, that webpage is sending a HTTP request to a server. That server in turn finds the HTML doc and sends it back to the user's browser. The web browser then interprets this HTML.

HTML

HTML stands for Hyptertext Markup Language.HTML is made up of text content, which is what you see, and mark up, which is what that content looks like.

Tags and Elements

HTML is made up of elements. Tags are then used to define these elements. An attribute is an element modifier. The anchor attribute is utilized to form links within elements.

Block and Inline: What's the difference?

Inline tags differ from block tags because block tags form an invisible box around the content inside of them.
